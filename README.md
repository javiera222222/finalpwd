Ice Cream Shop Project – Web Application with OOP and API
(versión en español abajo)
This project is a web application for an ice cream shop, where available products are displayed and managed through a backend API.
It was developed as part of the Object-Oriented Programming (OOP) course, Applying the theoretical concepts learned in the course: objects, inheritance, polymorphism, and database access.



 Technologies Used
 Backend

PHP 8+
Object-Oriented Programming
PDO for database access
REST API
MySQL / MariaDB

 Frontend

HTML5, CSS3, JavaScript

Fetch API or Axios for API communication

Proyecto Heladería – Aplicación Web con Programación Orientada a Objetos y API


Este proyecto consiste en una página web para una heladería, donde se muestran los productos disponibles y se puede interactuar con el backend mediante una API REST.
Fue desarrollado como trabajo integrador para la materia Programación Orientada a Objetos (POO), aplicando los conceptos teóricos de la materia: objetos, herencia, polimorfismo y acceso a datos.

 Tecnologías utilizadas
 Backend

PHP 8+
Programación Orientada a Objetos
PDO para acceso a base de datos
API REST
MySQL / MariaDB

 Frontend

HTML5, CSS3 y JavaScript

Fetch API o Axios para comunicación con el backend
